## ¿Qué es Spai?

Spai es una plataforma educativa que genera automáticamente artículos en español para estudiantes de idiomas. Usamos tecnología avanzada para crear contenido original adaptado a diferentes niveles de aprendizaje.

## Metodología

### Multi-Source Synthesis
Cada artículo se crea sintetizando información de **3-5 fuentes diferentes** de noticias en español, incluyendo:
- El País, BBC Mundo, El Mundo
- La Nación, Clarín, CNN Español
- Wikipedia, Deutsche Welle, y más

Esto asegura:
- ✅ Contenido original (no copiado)
- ✅ Perspectivas múltiples
- ✅ Información verificada

### Niveles CEFR

**A2 - Elemental:**
- Vocabulario básico (1000 palabras más comunes)
- Presente simple principalmente
- Frases cortas y claras
- ~200 palabras

**B1 - Intermedio:**
- Vocabulario intermedio
- Tiempos mixtos (presente, pretérito, imperfecto)
- Estructuras más complejas
- ~300 palabras

### Quality Control

Todos los artículos pasan por un sistema de evaluación automática que verifica:
- Gramática correcta para el nivel
- Valor educativo
- Calidad del contenido
- Vocabulario apropiado

**Puntuación mínima:** 7.5/10

Solo publicamos artículos que superan este estándar de calidad.

## Frecuencia

Generamos **12 artículos nuevos cada día**:
- 3 ejecuciones diarias (2am, 10am, 6pm UTC)
- 4 artículos por ejecución
- Mezcla de niveles A2 y B1
- Temas de actualidad

## Tecnología

- **Backend:** Python 3.11, SpaCy NER, Trafilatura
- **LLM:** OpenAI GPT-4o (generación) y GPT-4o-mini (calidad)
- **Sitio:** Jekyll + GitHub Pages
- **Automatización:** GitHub Actions
- **Código abierto:** [GitHub](https://github.com/aizlabs/spai)
